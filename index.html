Front-End Development

Languages:

HTML, CSS, JavaScript.

Frameworks/Libraries:

React, Angular, Vue.js.

Tasks:

Designing the layout, ensuring responsive design, creating interactive elements, and ensuring smooth user experience.

Back-End Development:

Languages:

Python, PHP, Ruby, Java, Node.js.

Frameworks

Django, Laravel, Express.js, Spring.

Tasks:

Handling server-side logic, databases, and application programming interfaces (APIs).

Full-Stack Development:

A full-stack developer works on both the front-end and back-end, covering the entire development process.

Web Design:

Focuses on the visual aspect and user interface (UI) of the website. Tools: Adobe XD, Figma, Sketch.

Database Management:

Databases:

MySQL, PostgreSQL, MongoDB, SQLite. Tasks include managing data storage, retrieval, and updates.

Version Control:

Tools

Git, GitHub, GitLab. Used to track changes in code and collaborate with other developers.

Web Hosting:

Websites are hosted on servers to be accessible on the internet. Common hosts: AWS, Heroku, Netlify, GoDaddy.
